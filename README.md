# Poker(斗地主)
## 项目描述
* 大四上程序作业
* 使用QT6进行设计（C/S架构设计）

## 构建环境
* Win10 x64
* QT6.6

## 使用方法
* 服务端与UI相连，点击Server.exe即可打开。
* 服务端使用文件系统存储文件，无需设置数据库。
* 同时打开服务端和客户端，就应该能动，

## 项目功能
* 登录注册
* 用户信息设置
* 创建游戏房间
* 斗地主游戏（未完善）
